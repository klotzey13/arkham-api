# Arkham Horror LCG API

A GraphQL API for Arkham Horror: The Card Game data, built with .NET 8, HotChocolate, and PostgreSQL.

**This is a personal learning project for practicing .NET development.**

## Tech Stack

- .NET 8 / ASP.NET Core
- GraphQL (HotChocolate)
- Entity Framework Core
- PostgreSQL
- Docker

## Getting Started

```bash
# Start PostgreSQL
docker-compose up -d

# Run the API
dotnet run
```

GraphQL Playground: `http://localhost:5095/graphql`
